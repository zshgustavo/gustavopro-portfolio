Analytics dashboard for monitoring business metrics.

## Overview 

- The system consists of five agents working in tandem:

- **Collector Agent (DOU-Collector)**: Responsible for accessing the official DOU website and extracting raw publication content.
- **Processor Agent (DOU-Processor)**: Transforms raw content into structured data using natural language processing techniques.
- **Organizer Agent (DOU-Organizer)**: Compiles the data into CSV format with structured columns.
- **Searcher Agent (DOU-Searcher)**: Enables efficient retrieval of specific information within the organized data.
- **Coordinator Agent (DOU-Coordinator)**: Manages the workflow between all other agents.
